## Requirements and Usage

For the Hid2BLE Stick, the XIAO ESP32-s3 is combined with a [USB-A Breakout](https://www.otronic.nl/en/usb-a-breakout-board.html) board which provides the 5V power supply. The boards are sticked together upside down using double-sided adhesive tape. Connect the GNDs and connect VUSB (5V) to 5V of the XIAO board. Power supply works from a USB powerbank or wall Adapter.
